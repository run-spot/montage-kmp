## 📦 Installation

Montage is distributed via **Maven Central**.

### 1️⃣ Add Maven Central Repository

`settings.gradle.kts`

```kotlin
dependencyResolutionManagement {
    repositories {
        google()
        mavenCentral()
    }
}
```

### 2️⃣ Add Dependency

`build.gradle.kts`

```kotlin
dependencies {
    implementation("run.thespot.montage:montage-kmp:0.0.3")
}
```

Current coordinates:

```text
groupId: run.thespot.montage
artifactId: montage-kmp
version: 0.0.3
```
